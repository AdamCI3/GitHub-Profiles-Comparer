# AngularGithubProfilesComparer

## User Stories

```

As a website visitor
I want to insert up to two gitHub names
So that I can  see information about them

As a website visitor
I want to compare two gitHub accounts
So I can see which one is better

As a website visitor
I want to add an account to a leaderboard
So I can see more account compared

As a website visitor
I want to sort the leaderboard
So I can see which account is the best in given category

As a website visitor
I want to switch beetwen compare section and leaderboard
So I can use both functionality
```
